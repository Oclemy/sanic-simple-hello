# Sanic Hello World

The simplest possible API using the [Sanic](https://sanic.dev) framework for Python.

## Requirements

- Python 3.8+

## Setup

```bash
pip install -r requirements.txt
```

## Run

```bash
python app.py
```

## Try it

Open your browser or use curl:

```bash
curl http://localhost:8080/
```

**Response:**

```json
{"message": "Hello, World!"}
```
